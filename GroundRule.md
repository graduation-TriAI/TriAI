**< 기본 규정 >**
1. 연락 빨리 확인하기
    - 읽씹 금지
    - 일상 생활 중에도 잘 확인하기
    - 하루 이내에는 무조건 확인하기
2. 제출물 데드라인 잘 지키기
    - 최종 제출물은 노션 or GitHub or 공유 드라이브에 업로드하기
3. 책임감 가지고 임하기
4. 서로의 의견을 경청하고, 피드백 솔직하게 공유하기
5. 개인 업무 잘 해오기
    - 개인 업무 데드라인 준수가 어려울 시 회의 하루 전까지 사유와 함께 카톡에 연락
6. Dataset, GitHub, 공유 드라이브 변경 사항 발생 시 바로 카톡에 연락

**< 회의 규정 >**
1. 회의 지각 금지
    - 회의 시간으로부터 5분 이후 지각으로 판단
    - 무단 지각 - 벌금 2000원 + 카톡에 연락
    - 사유 지각 - 카톡에 연락
2. 정기 회의 일시:
    - 강의 중 회의 - 월요일 오후 5시\~6시 15분, 목요일 오후 3시 30분\~6시 15분
    - 추가 회의 - 금요일 오후 8시\~10시 (대면/비대면을 유동적으로 조정하나 최대한 대면을 지향)
3. 회의 불참 규정:
    ※ 2명 이상 불참 - 해당 주차만 회의 일시 조정
    - 무단 불참 - 벌금 5000원, 회의록 확인 후 자신의 의견 notion에 기록 + 카톡에 연락
    - 사유 불참 - 회의록 확인 후 자신의 의견 notion에 기록 + 카톡에 연락
